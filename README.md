# typeScript-assignment-elzero_web_school-This repository includes solutions for all TypeScript tasks provided by the engineer Osama Elzero.
These tasks cover a variety of topics, including basic TypeScript syntax, variable and function typing, conditional types, and more advanced features
such as generic types and type inference. Each task is solved in a separate file , and the solutions are commented for better understanding.
This repository can be a valuable resource for anyone interested in learning TypeScript, as it provides real-world examples and practical exercises 
to reinforce the concepts covered in tutorials and documentation.
